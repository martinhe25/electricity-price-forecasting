# Electricity Price Forecasting (ERCOT)

This project explores forecasting hourly electricity prices in the ERCOT market using statistical and deep learning methods.

## Overview
Electricity prices are highly volatile and influenced by demand, weather, and grid conditions. This project studies time series forecasting approaches for predicting hourly locational marginal prices (LMPs).

## Data
- ERCOT hourly LMP data
- Weather variables such as temperature, humidity, and wind
- Historical data from 2018–2023

## Methods
- ARIMA
- LSTM
- Feature engineering for seasonality and lag structure

## Evaluation
- MAE
- RMSE
- Performance on price spikes vs. normal periods
